# Claude Project Instructions

Read `AGENTS.md` before doing any work in this repository.

`AGENTS.md` is the binding project contract. Follow it over this file when there is overlap.

Required behavior:

- Re-read `AGENTS.md` at the start of each implementation session.
- Check for child `AGENTS.md` files in the directories being edited.
- Keep all durable documentation under `docs/` except the allowed root files.
- Update `docs/developers/task-logs.md` after every meaningful task, fix, feature, or documentation change.
- Use `docs/plan/specs.md`, `docs/plan/design.md`, and `docs/plan/tasks.md` for planned work.
- Run relevant checks before committing.
- Do not add AI signatures, co-author trailers, generated-by notes, or tool metadata.
- GPG signing is not required.

If a task conflicts with `AGENTS.md`, stop and ask the project owner for direction.
